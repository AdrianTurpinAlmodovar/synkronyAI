# synkronyAI
SynkronyAI es una agencia de automatización e IA que optimiza empresas mediante flujos inteligentes. Combina un frontend dinámico en PHP/MySQL con animaciones GSAP y un ecosistema autónomo en Make.com. Usa modelos Llama 3 para clasificar leads, gestionar citas y alimentar un CMS de noticias desde Google Sheets y Botpress.
